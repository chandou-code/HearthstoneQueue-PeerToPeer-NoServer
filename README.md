# HearthstoneQueue-PeerToPeer-NoServer

## 自动化《炉石传说》排队工具

### 功能概述
通过监测在线笔记本网站 [http://www.xnote.cn/note/](http://www.xnote.cn/note/) 个人主页上第一个帖子的标题变化，自动开启《炉石传说》的排队功能。

### 使用步骤

1. **注册账号**
   在网站 [http://www.xnote.cn/note/](http://www.xnote.cn/note/) 上注册一个账号。

2. **配置环境**
   确保你的计算机上已安装 Python。  
   在命令行中输入以下命令以安装所需的依赖：
